# Architecture
* Use django bolt for APIs
    * General documentation can be found at https://github.com/dj-bolt/django-bolt/tree/master/docs/src/topics
    * API reference can be found at https://github.com/dj-bolt/django-bolt/tree/master/docs/src/ref
* Use React for frontend
    * Use Tanstack query and zustand for server state syncing and client state management
    * Use Tailwind for styling
* Use Procrastinate for background processing
    * Docs can be found at https://github.com/procrastinate-org/procrastinate/tree/main/docs
* Use Opik for LLM tracing and observability
* Use Qdrant as vector store
